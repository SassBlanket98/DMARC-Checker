IP Checker Help & Information

About IP Checker

IP Checker allows you to retrieve detailed information about any IP address, including:

  • Geolocation - City, region, and country associated with the IP
  • Network Information - ISP, ASN, and connection details
  • IP Reputation - Security and trustworthiness indicators
  • Technical Details - IP version and other characteristics

How to Use This Tool

  1. Enter an IP address in the input field, or leave it empty to check your current IP.
  2. Click "Check IP" to retrieve information about the specified IP address.
  3. Alternatively, click "Check My IP" to instantly check your current public IP address.
  4. View the detailed information, including geolocation, network details, and reputation.

IP Types

There are two main types of IP addresses:

  • IPv4 - The original IP address format (e.g., 192.168.1.1) consisting of four sets of numbers separated by periods.
  • IPv6 - The newer IP address format (e.g., 2001:0db8:85a3:0000:0000:8a2e:0370:7334) designed to overcome IPv4 address exhaustion.

Privacy Information

When using the "Check My IP" feature, the tool retrieves your public IP address - the identifier that your internet service provider assigns to your internet connection. This is the IP address that websites and online services see when you connect to them.

We do not store IP addresses or any information associated with them. All lookups are performed in real-time and the data is not saved to any database.
