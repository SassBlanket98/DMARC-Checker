IP Types

There are two main types of IP addresses:

  • IPv4 - The original IP address format (e.g., 192.168.1.1) consisting of four sets of numbers separated by periods.
  • IPv6 - The newer IP address format (e.g., 2001:0db8:85a3:0000:0000:8a2e:0370:7334) designed to overcome IPv4 address exhaustion.

Understanding Reputation Scores

Our weighted reputation scoring system provides clear threat assessment:

  • 80-100: Excellent - Clean reputation across all sources
  • 60-79: Good - Generally safe with minor cautions
  • 40-59: Moderate - Some risk indicators present
  • 0-39: Poor - Significant security concerns detected

Scores are calculated by analyzing data from all external sources, DNSBL listings, and historical threat patterns.

Privacy & Performance Information

When using the "Check My IP" feature, the tool retrieves your public IP address - the identifier that your internet service provider assigns to your internet connection. This is the IP address that websites and online services see when you connect to them.

Enhanced Security: We query multiple external threat intelligence sources (AbuseIPDB, VirusTotal, and 18+ DNSBL services) to provide comprehensive security analysis. All queries are performed securely over HTTPS.

Privacy Protection: We do not store IP addresses or any information associated with them. All lookups are performed in real-time and the data is not saved to any database. External API calls are made directly from our servers to protect your privacy.

Performance: Results are cached temporarily for improved performance, and concurrent API processing ensures fast response times even when querying multiple threat intelligence sources.
